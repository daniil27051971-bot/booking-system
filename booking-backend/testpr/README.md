# Booking System Backend

## Описание проекта

Система бронирования ресурсов организации, разработанная на Python с использованием FastAPI и PostgreSQL.

Проект предназначен для автоматизации процесса бронирования помещений, оборудования и рабочих мест. Пользователи могут создавать, изменять и отменять бронирования, а администраторы управляют ресурсами и контролируют работу системы.

---

## Технологический стек

### Backend

* Python 3.14
* FastAPI
* SQLAlchemy
* PostgreSQL

### Безопасность

* JWT-аутентификация
* Хранение паролей в виде хешей
* Ролевая модель доступа (User / Admin)

### Тестирование

* Pytest

## Информационная безопасность

В проекте реализованы следующие меры защиты:

* хранение паролей в виде хешей;
* использование JWT-токенов;
* разграничение прав доступа;
* защита административных операций;
* аудит действий пользователей;
* использование переменных окружения для хранения конфиденциальных данных.

Конфиденциальные параметры не размещаются в исходном коде и хранятся в файле `.env`.

## Установка

### Клонирование репозитория

```bash
git clone <repository_url>
cd booking-backend
```

### Создание виртуального окружения

```bash
python -m venv venv
```

### Активация окружения

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Настройка переменных окружения

Создать файл `.env`:

```env
DATABASE_URL=postgresql://postgres:password@localhost:5432/booking
SECRET_KEY=your_secret_key
```

### Создание базы данных

Выполнить SQL-скрипт:

```bash
booking.sql
```

---

## Запуск проекта

```bash
uvicorn main:app --reload
```

После запуска документация API будет доступна по адресу:

```text
http://127.0.0.1:8000/docs
```
